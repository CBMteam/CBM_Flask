# CBM_Flask
저혈당 예측 및 서비스 제공 앱을 위한 flask server
<br>

```
✨ 저혈당 예측 결과 전달

✨ 인슐린/탄수화물 입력 받아서 시계열 데이터 update
```

### 📒 Main Function
- **저혈당 예측 결과 전달**
  - 데이터 전처리
  - 서버에 저장된 머신 러닝 모델 이용해서 다음 저혈당까지 남은 시간 반환
	
- **인슐린/탄수화물 입력 받아서 시계열 데이터 update**
	- 인슐린/탄수화물 데이터 입력 받기
  - 시계열 데이터로 변환해서 MySQL에 저장

### 📙 사용된 도구

* [Flask]
* [MySQL]
